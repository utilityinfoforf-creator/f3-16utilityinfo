# Installing the Utility Dashboard

Your app is now a **Progressive Web App (PWA)** and can be installed on mobile devices and desktops!

## 📱 Mobile Installation

### Android
1. Open the app in **Chrome** or **Edge** browser
2. Look for the **"Install" (⬇️)** button at the top of the dashboard, or tap the browser menu (⋮)
3. Select "Install app"
4. Tap "Install" to add to home screen
5. The app will appear as an icon on your home screen

### iPhone (iOS)
1. Open the app in **Safari**
2. Tap the **Share button** (square with up arrow)
3. Scroll down and tap **"Add to Home Screen"**
4. Give it a name (e.g., "Utility Portal")
5. Tap **"Add"**
6. The app will appear as an icon on your home screen

## 🖥️ Desktop Installation

### Windows/Mac/Linux
1. Open the app in **Chrome**, **Edge**, or **Opera**
2. Look for the **install icon** (looks like a box with download arrow) in the address bar
3. Click it and select "Install"
4. The app will open in a window like a native application
5. Shortcuts will be available on your desktop or start menu

## ⚙️ Features

- **Standalone mode** - Runs without browser UI (clean, app-like experience)
- **App icon** - Appears on home screen or desktop
- **Offline support** - Connection errors are handled gracefully
- **Custom theme** - Purple theme that matches your branding
- **Quick shortcuts** - Long-press app icon to access popular actions:
  - Make Payment
  - View Bills

## 🚀 How It Works

- Uses **Web App Manifest** (`manifest.json`) to define app properties
- **Service Worker** handles installation and network requests
- **No app store needed** - Installs directly from your website
- **Works across all devices** - Same app on mobile, tablet, and desktop

## 📋 Requirements

- HTTPS connection (required for PWAs)
- Modern browser (Chrome, Edge, Opera, Safari, Firefox)
- Active internet connection

## 🐛 Troubleshooting

**Install button not showing?**
- Make sure you're using a supported browser (Chrome, Edge, Opera)
- Try clearing browser cache and reloading
- Some browsers only show install prompt after visiting the site multiple times

**App not working after install?**
- App requires internet connection (online-only mode)
- Check your network connection
- Reinstall the app if needed

**How to uninstall?**
- **Mobile**: Long-press the app icon → Remove/Uninstall
- **Desktop**: Right-click app → Uninstall or move to trash
